=== *에러코드*

==== *형식*

<<link-error-reference, Reference>> 에서는 `이름(에러코드, 기본 메세지)` 형식으로 기술합니다. +
서버 내부 오류 발생 시, Response Body 의 error 는 아래와 같이 리턴됩니다. +

```jsx
error : {
    code : 5000000,
    message : '서버 내부 오류'
}
```

code : 에러코드는 정수형으로 리턴됩니다. +
message : `기본 메세지 + 상세 메세지` 처럼 `기본 메세지` 에 `상세 메세지` 가 추가되어 리턴될 수 있습니다. +
